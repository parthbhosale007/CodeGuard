# 🛡️ CodeGuard – AI-Powered CI/CD Code Reviewer

CodeGuard is an AI-integrated CI/CD system that automatically reviews code, detects bugs and security issues, and provides actionable suggestions directly in pull requests.

## 🚀 Features

* AI-based code review
* Automated CI/CD pipeline using GitHub Actions
* Security issue detection (DevSecOps approach)
* Smart failure analysis with fix suggestions

## 🛠️ Tech Stack

* Python (FastAPI)
* GitHub Actions
* OpenAI API (LLM)

## 🎯 Goal

To automate code review and enhance software quality using AI-driven DevOps practices.
